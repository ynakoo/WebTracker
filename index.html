1 Download Extension

To use the extension locally, you need to download the source files. The components have already been initialized and built for your convenience.

zipFile Here

2 Installation Steps

  • Open Chrome Extensions

    In your Chrome browser, type chrome://extensions/ in the address bar and press Enter.

  • Enable Developer Mode

    Toggle the "Developer mode" switch in the top right corner of the extension management page.

  • Load Unpacked Extension

    Click the "Load unpacked" button and select the WebTracker folder (the one containing manifest.json).

3 Initialisation Commands

If you wish to rebuild the extension components from source, use the following commands:

# Build POPUP component

cd POPUPP && npm install && npm run build

# Build DASHBOARD component

cd DASHBOARD && npm install && npm run build
